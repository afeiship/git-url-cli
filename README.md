# git-url-cli
> Get git http/https/ssh/git url.

## installation
```shell
# install
npm i -g @feizheng/git-url-cli

# uninstall
npm uninstall -g @feizheng/git-url-cli
```

## usage
```shell
guc
# https://github.com/afeiship/git-url-cli

# open the `url`
open `guc`
```
