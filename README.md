# git-url-cli
> Get git http/https/ssh/git url.

## installation
```shell
npm i -g @feizheng/git-url-cli
```

## usage
```shell
git-url-cli

# https://github.com/afeiship/git-url-cli

# open the `url`
open `git-url-cli`
```
